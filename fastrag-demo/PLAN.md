# AIS Mock 演示系统 - 分步实施计划

## 总览

共 12 个阶段，每阶段完成后均可独立验证。

## ✅ 阶段 1：项目脚手架搭建（已完成）
**目标**：初始化 Vue3 + TS + Vite 项目，安装核心依赖

**操作**：
1. 使用 Vite 创建 Vue3 + TypeScript 项目
2. 安装依赖：vue-router, pinia, element-plus, axios, sass, echarts
3. 配置 vite.config.ts（路径别名、Element Plus 自动导入）
4. 配置 tsconfig.json
5. 创建基础目录结构

**验证**：`npm run dev` 能启动，浏览器访问看到空白首页

## 实施策略

- 每个阶段完成后运行 `npm run dev` 验证
- 每个阶段的代码提交一次 git commit
- Mock 数据在阶段 3 统一初始化骨架，后续阶段逐步补充
- 复杂组件优先实现核心功能，交互细节后续迭代
