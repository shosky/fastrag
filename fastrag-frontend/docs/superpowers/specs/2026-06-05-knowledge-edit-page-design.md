# 知识库编辑页面设计文档

## 概述

为知识库管理系统添加编辑功能，允许用户修改已创建知识库的配置信息。采用与创建页面复用的架构设计，同时支持多种文件类型（文档、音频、视频）。

## 目标

- 支持编辑知识库的基础信息（名称、简介、分类、标签、权限）
- 支持配置允许的文件类型
- 保持与创建页面一致的 UI/UX
- 提供危险操作区（删除知识库）

## 组件设计

### form.vue 通用表单

**Props：**
```typescript
interface Props {
  mode: 'create' | 'edit'
  initialData?: KnowledgeBase  // 编辑模式时传入
}
```

**核心逻辑：**
- 根据 `mode` 决定显示/隐藏特定字段
- 编辑模式下预填充表单数据
- 编辑模式下嵌入模型字段设为只读
- 编辑模式下右侧显示操作区而非进度区

### edit.vue 编辑入口

**职责：**
- 从路由获取知识库 ID
- 加载知识库数据
- 将数据传递给 form.vue
- 处理保存成功后的跳转

## 数据流

```
edit.vue
  │
  ├─ 1. 获取路由参数 id
  ├─ 2. 调用 API 加载知识库数据
  ├─ 3. 传递 initialData 给 form.vue
  │
  └─ form.vue
      ├─ 4. 预填充表单
      ├─ 5. 用户编辑
      ├─ 6. 表单验证
      └─ 7. 调用保存 API
            └─ 8. 返回列表页
```

## 交互设计

### 保存流程
1. 点击「保存修改」按钮
2. 表单验证
3. 调用更新 API
4. 显示成功提示
5. 跳转回知识库详情页

### 删除流程
1. 点击「删除知识库」按钮
2. 弹出二次确认对话框
3. 输入知识库名称确认
4. 调用删除 API
5. 跳转回知识库列表页

### 取消操作
1. 点击「取消」按钮
2. 检测是否有未保存的修改
3. 如有修改，弹出确认对话框
4. 返回知识库详情页
