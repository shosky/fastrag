# 日志管理设计文档

## 1. 功能概述

日志管理记录知识库的操作日志和变更日志，支持审计追溯。

## 2. 数据模型

### 2.1 数据库表 `kb_log`

| 字段 | 类型 | 说明 |
|------|------|------|
| id | VARCHAR(32) PK | 日志ID |
| kb_id | VARCHAR(32) FK | 知识库ID |
| category | VARCHAR(16) | 分类(operation/retrieval/publish) |
| action | VARCHAR(64) | 操作 |
| target | VARCHAR(256) | 操作对象 |
| detail | TEXT | 详情 |
| operator | VARCHAR(64) | 操作人 |
| timestamp | DATETIME | 时间 |

### 2.2 数据库表 `kb_update_log`

| 字段 | 类型 | 说明 |
|------|------|------|
| id | VARCHAR(32) PK | 日志ID |
| kb_id | VARCHAR(32) FK | 知识库ID |
| update_type | VARCHAR(32) | 变更类型 |
| target | VARCHAR(256) | 变更对象 |
| detail | TEXT | 详情 |
| operator | VARCHAR(64) | 操作人 |

## 3. API 接口

### 3.1 操作日志
```
GET /api/kb/{kbId}/logs
```

**查询参数：**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| category | string | 否 | 分类筛选(operation/retrieval/publish) |

### 3.2 变更日志
```
GET /api/kb/{kbId}/update-logs
```

**查询参数：**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| type | string | 否 | 变更类型筛选 |
