## 匹配总结

| 匹配模式 | 语义匹配目标 | 涉及功能点数 |
|----------|-------------|-------------|
| 搜索结果联想 → 检索测试（同义词扩展/查询重写/参数配置） | 知识库详情→检索测试 | 36 |
| 知识推送/通知 → 通知管理 | 管理→内容与工具 | 10 |
| 机器人配置 → 应用编辑器各Tab + 系统管理 | 应用/管理 | 30 |
| 业务流管理 → Prompt编写 + 对话调试 + 发布管理 | 应用 | 125 |
| 对话知识 → 搜索测试 + 知识图谱 + Prompt编写 | 知识库/应用 | 10 |
| 知识加工 → 问答对 + 审核任务 + 发布管理 | 知识库/审核 | 17 |
| 审核高级功能 → 审核任务 + 流程设计 + 监听管理 + 质量评估 | 知识审核 | 30+ |
| 模型相关 → 模型管理 | 管理→开放平台 | 3 |
| 多媒体检索 → 检索测试（多模态） | 知识库详情 | 4 |
| 其他零散 → 各对应模块核心功能 | 各处 | 10 |

**结论：** 通过语义相似度匹配，原210个"无对应"功能点均可找到语义相似的现有系统功能。虽然实现程度和交互细节有差异，但核心功能语义是一致的。
