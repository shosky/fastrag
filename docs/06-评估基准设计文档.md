# 评估基准设计文档

## 1. 功能概述

评估基准是用于 RAG 评估的标准问题集，支持手动创建和 AI 自动生成。

## 2. 数据模型

### 2.1 数据库表 `kb_benchmark`

| 字段 | 类型 | 说明 |
|------|------|------|
| id | VARCHAR(32) PK | 基准ID |
| kb_id | VARCHAR(32) FK | 知识库ID |
| name | VARCHAR(128) | 名称 |
| description | TEXT | 描述 |
| has_gold_chunks | TINYINT | 是否有标准切片 |
| has_gold_answer | TINYINT | 是否有标准答案 |
| is_auto_generated | TINYINT | 是否自动生成 |
| question_count | INT | 问题数量 |

### 2.2 数据库表 `kb_benchmark_question`

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT PK | 问题ID |
| benchmark_id | VARCHAR(32) FK | 基准ID |
| question | TEXT | 问题 |
| gold_chunks | VARCHAR(512) | 标准切片 |
| gold_answer | TEXT | 标准答案 |

## 3. API 接口

### 3.1 基准列表
```
GET /api/kb/{kbId}/benchmarks
```

### 3.2 基准详情
```
GET /api/kb/{kbId}/benchmarks/{id}
```

### 3.3 创建基准
```
POST /api/kb/{kbId}/benchmarks
```

### 3.4 自动生成
```
POST /api/kb/{kbId}/benchmarks/generate
```

### 3.5 删除基准
```
DELETE /api/kb/{kbId}/benchmarks/{id}
```
