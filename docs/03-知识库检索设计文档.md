# 知识库检索设计文档

## 1. 功能概述

知识库检索是 RAG 的核心功能，支持全文检索、向量检索、混合检索等多种模式。

## 2. API 接口

### 2.1 检索搜索

```
POST /api/retrieval/search
```

**请求体：**
```json
{
  "knowledgeId": "kb_sample",
  "query": "搜索关键词",
  "config": {
    "mode": "hybrid",
    "topK": 10,
    "similarityThreshold": 0.5,
    "enableRerank": true,
    "rerankModel": "bge-reranker-v2-m3"
  }
}
```

**响应：**
```json
{
  "code": 200,
  "data": [
    {
      "index": 1,
      "similarity": 0.85,
      "content": "匹配的切片内容",
      "source": "文档名称",
      "fileId": "file_id",
      "chunkIndex": 0,
      "distance": 0.15,
      "highlights": ["匹配的关键词"]
    }
  ]
}
```

### 2.2 获取切片数量

```
GET /api/retrieval/kb/{kbId}/chunks/count
```

### 2.3 查询建议

```
POST /api/query/suggest
```

### 2.4 同义词扩展

```
POST /api/query/expand-synonyms
```

### 2.5 查询规则应用

```
POST /api/query-rules/apply
```

**请求体：**
```json
{
  "query": "原始查询"
}
```

### 2.6 图谱扩展

```
POST /api/graph/expand
```

**请求体：**
```json
{
  "kbId": "知识库ID",
  "query": "查询关键词",
  "depth": 2,
  "maxEntities": 20
}
```

## 3. 检索模式

| 模式 | 说明 | 实现状态 |
|------|------|----------|
| 全文检索 | Elasticsearch 全文匹配 | ✅ 已实现 |
| 向量检索 | Milvus 向量相似度搜索 | ⚠️ 已有服务，未接入检索 |
| 混合检索 | 全文+向量融合 | ❌ 未实现 |
| 图谱检索 | Neo4j 图谱扩展 | ✅ 已实现 |
